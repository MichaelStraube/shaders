# shaders
Collection of Shadertoy shaders displayed in a Qt window.

## Build
```
mkdir build
cd build
cmake ../ -DCMAKE_BUILD_TYPE=Release
make
```

## Usage
You can skip through the shaders with the mouse wheel.
